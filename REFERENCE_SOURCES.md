# Reference Sources

The v1 vocabulary is deterministic field-survey reference data compiled for preservation documentation workflows. It is intended as a starting controlled vocabulary, not a substitute for current SHPO guidance.

## Core Standards

- National Park Service, National Register Bulletin 15, *How to Apply the National Register Criteria for Evaluation*. Used for the seven aspects of integrity and resource type framing.
- Secretary of the Interior's Standards for the Treatment of Historic Properties, especially the Standards for Rehabilitation. Used for alteration and treatment language references.
- National Park Service Preservation Briefs 1-47. Used as preservation treatment context for materials, windows, roofing, masonry, storefronts, and related conditions.
- HABS/HAER/HALS documentation guidance. Used for field documentation emphasis on description, location, condition, and materials.
- Getty Art & Architecture Thesaurus. Used as a vocabulary model for architectural style, material, and feature terms.
- National Register historic function/use categories. Used to seed historic and current use dropdown terms.
- State Historic Preservation Office public program pages. Used to seed jurisdiction names and links. Confirm current submission forms and survey requirements directly with each SHPO before official filing.

## Notes

The lookup tables favor conservative, commonly recognized U.S. field survey terms. Local governments and SHPOs often require more specific vocabularies, eligibility codes, or inventory IDs; the generated reference tabs are intentionally safe to customize after initialization.
